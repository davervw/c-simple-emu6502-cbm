Commodore 64 Keyboard Emulator, key scan codes helper

  1. Flash LilyGo T-Display-S3 with C64 Emulator via USB cable, with option USB CDC On Boot: "Enabled"
  2. Browser to Serial Port
  3. While keeping this page in focus, type to get keystrokes in emulator
  4. Optionally open Dev Tools to watch real-time codes in console log
  5. Repeat from step 2 if disconnnected, if trouble power/reset board.
